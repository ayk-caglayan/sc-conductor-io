# Sample audio (optional)

Place reference audio here for presets that use `preloadBuffers`, for example:

- `Celesta_F.wav`
- `Celesta_F_aug5.wav`

In preset files, use paths relative to the repo root, e.g. `samples/Celesta_F.wav`.

For the bundled `grant_demo.preset`, no samples are required — it uses live microphone input only.

If you are working from the [Celesta_piece](https://github.com/ayk-caglayan/Celesta_piece) composition repo, copy the celesta WAV files from that repository into this folder.
